# 내일뭐입지?, what2wear

java와 javafx를 활용한 옷장 관리 및 AI 코디 추천 서비스, what2wear의 레포지토리입니다.

## 📌 프로젝트 개요
> 단순히 사용자의 옷을 기반으로 코디를 저장하는 기능 뿐 아니라 날짜 별 코디를 기록하고, 커뮤니티를 통해 스타일링을 공유 및 참고하기, AI의 도움을 받아 맞춤형 코디를 자동으로 생성하는 기능 제공
