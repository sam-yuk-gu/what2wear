# 👕 내일뭐입지?, what2wear

java와 javafx를 활용한 옷장 관리 및 AI 코디 추천 서비스, what2wear의 레포지토리입니다.

## 📌 프로젝트 개요
> what2wear는 Java와 JavaFX를 기반으로 제작된 옷장 관리 및 AI 코디 추천 서비스입니다.<br />
사용자는 자신의 옷을 등록하고 이를 바탕으로 코디를 저장할 수 있으며, 날짜별로 착장 기록을 남길 수 있습니다.<br />
또한, 커뮤니티를 통해 스타일링을 공유하거나 참고할 수 있고, AI 기능을 통해 날씨와 상황에 맞는 맞춤형 코디를 자동으로 추천받을 수 있습니다.
